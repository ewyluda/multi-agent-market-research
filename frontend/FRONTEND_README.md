# Multi-Agent Market Research - Frontend

React frontend for the AI-powered stock market analysis platform.

## Quick Start

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Open http://localhost:5173
4. Make sure backend is running on port 8000

## Features

- Real-time analysis updates via WebSocket
- Interactive dashboard with 5 specialized agents
- BUY/HOLD/SELL recommendation gauge
- Sentiment analysis breakdown
- News feed and price charts
- Dark theme UI

## Usage

1. Enter a ticker (e.g., NVDA)
2. Click "Run Analysis"
3. Watch agents work in real-time
4. View comprehensive results

See main README.md for full documentation.
